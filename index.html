Import Format Guide
JSON format is the most complete — exports and re-imports the entire price database including all groups and shades.

To export the current price list as JSON, go to Price List → Export JSON. You can then edit the file and re-import it. The structure is:

{
  "priceDB": {
    "master": {
      "1": { "groups": [ { "label":"Gr I", "shades":"Base White, Ultra White", "prices": {"4L":760,"1L":193,"500ml":107} } ] },
      "10": { "simple": { "35kg":3415, "7kg":810 } }
    },
    "bj": { ... },
    "fixed": { ... }
  },
  "productsDB": [
    { "id":1, "name":"Raunaq Super Synthetic Enamel", "type":"dpl", "category":"enamel", "bihar":"yes" }
  ]
}
CSV is useful for bulk price updates. Each row updates one product's price for one pack size.

CSV file must have these exact columns (header row required):

section,productId,group,packSize,price
master,1,0,4L,760
master,1,0,1L,193
master,1,1,4L,725
bj,1,0,4L,760
fixed,13,0,20L,4100

section: master / bj / fixed  |  productId: numeric product ID  |  group: 0-based group index (use 0 for simple/first group)  |  packSize: 4L / 1L / 500ml etc  |  price: numeric price

Import your full RM list at once using this CSV format. Existing RM codes will be updated; new ones will be added.

RM Master CSV format (header row required):

code,name,price,unit,supplier
TIO2-001,Titanium Dioxide (Rutile),280,kg,Premium Pigments Ltd
ALK-001,Alkyd Resin (Long Oil),145,kg,Resin Corp
SLV-001,Mineral Turpentine Oil,95,ltr,Local Supplier

unit: kg / ltr / g  |  supplier: optional

📐 Costing Formula Reference
Per Litre Cost (Corrected)
Total Litres = Total RM Qty (kg) ÷ Wt per Litre
Total RM Cost = Σ (RM Price × Qty)
Mfg Cost = Total RM Cost × Mfg%
Total Cost = Total RM Cost + Mfg Cost
Per Litre = Total Cost ÷ Total Litres
Matches spreadsheet: G26 = D26 / H5
where D26 = Total Cost, H5 = Wt per Litre
Pack Cost & Profit
Paint Cost/Pack = Per Litre × Pack Litres
Final Cost/Pack = Paint + Container + Carton/n
Selling Price = DPL Rate × (1 − Discount%)
Profit/Pack = Selling Price − Final Cost
Profit% = Profit ÷ Selling Price × 100
DPL Rate comes from Price List page.
